
# Restaurant Menu & Order Management System

This project is developed as part of an assignment based on a predefined dataset.  
The program uses the given menu, inventory, and sales data to simulate restaurant operations.

## What this program does

- Displays menu items grouped by category  
- Handles cart operations like add, update, and remove  
- Calculates bill including GST  
- Tracks inventory and shows reorder alerts  
- Performs basic sales analysis  

## Concepts Used

- Lists and Dictionaries  
- Nested data structures  
- Loops and conditionals  
- Functions  
- Deep copy
