# File I/O, APIs & Exception Handling

This assignment is based on understanding how Python works with files, APIs, and error handling.  
The program is designed to perform basic file operations, fetch data from an API, and handle different types of errors without crashing.

## What the program does

- Creates a file and writes some Python notes into it  
- Appends additional content to the same file  
- Reads the file and allows keyword search  
- Fetches product data using an API  
- Filters and displays useful information  
- Handles invalid inputs safely  
- Stores errors in a separate log file  

## Files in this folder

- part3_api_files.py → main program  
- python_notes.txt → generated after running the code  
- error_log.txt → contains logged errors  
- README.md → this file  

## Note

The text files are created only after running the program.  
So make sure the code is executed completely before uploading.
