# Data Visualization & Machine Learning

This assignment focuses on analysing student performance data using Python. It includes data exploration, visualization, and building a simple machine learning model to predict whether a student will pass or fail.

## Overview

The dataset contains marks of students in different subjects along with attendance and study hours. Using this data, different visualizations are created and a classification model is trained.

## What is implemented

- Loaded and explored the dataset using pandas  
- Calculated basic statistics and analysed pass/fail performance  
- Created multiple visualizations using matplotlib (bar, histogram, scatter, box, line)  
- Used seaborn for better visual representation  
- Built a Logistic Regression model to predict student results  
- Evaluated model accuracy and analysed feature importance  

## Files included

- part4_visualization_ml.ipynb → main notebook  
- students.csv → dataset used  
- plot1_bar.png to plot7_seaborn_scatter.png → generated graphs  
- README.md → project description  

## Important note

All plots are generated using the notebook.  
The notebook is executed fully so that all outputs and graphs are visible.
